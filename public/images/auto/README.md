# Vantage Auto — featured vehicle images

Drop **verified** photos here using the EXACT filenames below. The filename is
the contract: the catalog (`lib/autoCatalog.ts`) points each card at one path,
and the card shows an "Image coming soon" placeholder until that exact file
exists. Never substitute a different model — a wrong photo is worse than no
photo.

| Filename                     | Must be a photo of                         | Body style |
| ---------------------------- | ------------------------------------------ | ---------- |
| `mercedes-g63.jpg`           | Mercedes-Benz G63 AMG (G-Class), 2020+     | SUV        |
| `mercedes-s-class.jpg`       | Mercedes-Benz S-Class (S580 / S63), 2022+  | Sedan      |
| `porsche-911.jpg`            | Porsche **911** (NOT Panamera), 2020+      | Coupe      |
| `toyota-corolla.jpg`         | Toyota Corolla S / SE, 2020+               | Sedan      |
| `chrysler-pacifica.jpg`      | Chrysler Pacifica Touring, 2020+           | Minivan    |
| `bmw-3-series.jpg`           | BMW 3 Series, 2023+                         | Sedan      |
| `mercedes-c300.jpg`          | Mercedes-Benz C300 (C-Class), 2023+        | Sedan      |
| `bmw-5-series.jpg`           | BMW **5 Series** sedan (NOT X5), 2023+     | Sedan      |
| `bmw-x5.jpg`                 | BMW **X5** SUV (NOT 5 Series), 2023+       | SUV        |

## Verify before publishing
- Porsche 911 card must NOT show a Panamera.
- BMW X5 card must NOT show a 5 Series, and vice-versa.
- No generic luxury-car photo under a specific model label.

## Recommended specs
- Landscape, ~16:11 aspect, ≥ 1200px wide, JPG with reasonable compression.
- Clean background, vehicle as the clear subject.
